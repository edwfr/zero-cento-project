# 📁 Implementation Documentation

Questa cartella contiene la documentazione operativa della fase di implementazione di ZeroCento.

## 🎯 Posizionamento Ufficiale

ZeroCento è una **training management platform trainer-led**:
- il trainer definisce schede, progressioni e decisioni tecniche;
- la piattaforma supporta esecuzione, raccolta feedback e monitoraggio;
- non è previsto un layer di coaching AI runtime come fonte decisionale primaria.

## 📌 Fonte Di Verità

Per evitare conflitti documentali, usare questi file in ordine:

1. **[CHECKLIST.md](CHECKLIST.md)** - stato task canonico (aperti/chiusi)
2. **[CHANGELOG.md](CHANGELOG.md)** - cronologia implementazioni
3. **[NEXT_ACTIONS.md](NEXT_ACTIONS.md)** - backlog attivo (solo task residui)
